# Base Trust Passport

### Your Wallet. Your Identity. Your Reputation.

Base Trust Passport is an on-chain identity and reputation protocol built on Base Mainnet.

It analyzes wallet activity to calculate a Trust Score, assign Passport Levels, issue a Soulbound Passport NFT, and unlock collectible Credential and Level NFTs.

---

## Features

- Trust Score (0–1000)
- Passport Levels
- Soulbound Passport NFT
- Credential NFTs
- Level NFTs
- Wallet Analytics
- AI Wallet Report


## NFTs

### Passport NFT

- ERC721
- Soulbound
- One Passport per Wallet

### Credential NFTs

- ERC1155
- Transferable
- Sequential Claim

### Passport Level NFTs

- ERC1155
- Transferable
- Sequential Claim

## Vision

Base Trust Passport aims to provide a portable on-chain identity that allows wallets to build verifiable reputation across the Base ecosystem.
